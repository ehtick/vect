About Diagram Syntax

Diagram syntax is in YAML

Data structure:

  • nodes is array of Node:
    • name: graph node identifier (could be referenced in edges from/to)
    • label: business name of the node
    • type: (default: stand) possible values: stand | compact | barrel. Will be enhanced.
    • icon: Material Icon
    • composition: if node belongs to composition (parent), then provide name of composition
  • compositions is array of Composition:
    • name: composition identifier (could be referenced from Node and Composition)
    • label: business name of the composition (eg Data Access Layer)
    • border: (default: plane) possible values: plane | frame. Will be enhanced.
    • composition: if composition is embedded into parent composition, then provide name of parent composition
  • edges is array of Edge:
    • from: name of Node (start of the connection)
    • to: name of Node (end of the connection)
    • name: identifier of edge. Very optional.
    • label: business label, which describes meaning of connection

Shortcuts

  • Ctrl+Enter: Save diagram source and draw diagram
  • Ctrl+E: Show/hide diagram editor
  • Ctrl+C: Center diagram
  • Ctrl+H: Diagram Help
